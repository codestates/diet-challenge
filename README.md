# FEATURE
* 목표를 정해서 원할 때마다 사진과 글을 업로드할 수 있습니다. 
* 목표에 도달하는 과정을 기록해서 자신의 활동 내역을 볼 수 있습니다.
* 같은 목표를 가진 사람들과 친구 신청을 통해 관계를 맺을 수 있습니다.
* 친구들과 각자의 목표와 진행상황을 사진과 글로 공유할 수 있습니다.
* 메인페이지에 자신과 친구들의 최신 활동을 바로 확인할 수 있습니다.
